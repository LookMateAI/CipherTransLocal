功能模块

把常用传输动作放在最短路径里。

这不是网盘，也不是复杂同步工具。它只专心处理一件事：让你在 Windows 和 Android 之间更快、更稳地把内容发过去。

局域网自动发现

两端打开后自动寻找同一网络里的设备，不用手动输入 IP，也不用扫一堆配置项。

消息、图片、文件一起传

临时文字、截图、照片、APK、压缩包都在一个会话里，传完之后也能回头找。

进度看得见

大文件传输时能看到速度、进度和状态。卡没卡住，不用猜。

暂停、继续、取消

传错文件可以取消，网络不稳可以重试，传输控制不藏在复杂菜单里。

Android 后台更稳

传输时使用前台服务和 Wi-Fi 保活策略，减少退到后台后连接中断的情况。

误删前会确认

删除设备、历史记录这类动作会二次确认，避免手滑把重要记录清掉。

真实界面

不是概念图，是真的能拿来传。

桌面端负责稳定收发和后台运行，Android 端把设备、历史和设置做成手机上更顺手的结构。

怎么用

打开两端，看到设备，直接发送。

它的使用方式尽量接近日常聊天：先选设备，再发消息或文件。你不需要记住 IP，也不需要登录账号。

1

两台设备连同一个网络

电脑和手机连接同一个 Wi-Fi，或者处在可互通的局域网里。

2

打开 CipherTransLocal

桌面端和 Android 端会互相广播并发现对方设备。

3

像聊天一样发送

发送文字、图片或文件，进度和结果都会留在对应设备的会话里。
